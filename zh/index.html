Macula

快速开始 GITHUB

Macula是基于Spring Boot和Spring Cloud Alibaba 的微服务开发框架及面向企业级的系统管理、权限、文档、网关和认证服务

Macula生态主要包含微服务开发框架macula-framework、基础系统macula-base、批处理macula-batch、认证服务macula-oauth2、基于KONG的API网关以及基于微前端的macula-portal

开发框架

macula-framework是基于spring-boot和spring-cloud-alibaba开发的用于单体或者微服务应用开发的框架，支持多数据源、多redis配置、分布式锁、统一异常处理、模型转换以及基于阿里巴巴微服务生态搭建的微服务治理体系

管理后台

macula-base是支撑系统运行的统一管理后台，包括系统管理、用户中心、权限管理、文档中心、配置管理、网关管理、服务治理、分布式事务、批处理管理以及低代码开发等功能

批处理

macula-batch是批处理集中调度中心，支持单任务和可视化的工作流任务配置

统一认证

macula-oauth2是认证服务，支持标准的Oauth2.0协议，包括支持密码、短信验证码、微信、企业微信、运营商手机号等多种登录方式

API网关

API网关是基于KONG二次定制的，集成了认证服务、权限控制等插件

微前端门户

macula-portal是作为业务后端的总入口而开发的，可以将多个微服务管理后端通过微前端的方式集成成一个整体的后端系统
